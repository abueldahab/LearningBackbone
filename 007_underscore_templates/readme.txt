------------------------------------------------------
   Learning Backbon - 007 - Underscore's Templates
------------------------------------------------------

This example purpose is to learn different ways Backbone and Undescore can works together to display
some data into the page.

Models stores data.
Models handles data changes and trigger events.

Views render things to some DOM nodes.
Often views inject HTML into DOM nodes.

TEMPLATES merges model's data with HTML structure to produce the "thing" to 
be injected into DOM nodes.
